IPL Match Analytics

Overview

This project analyzes IPL ball-by-ball data from the last five seasons to understand what contributes most to winning matches. Using Python and pandas, the project studies toss impact, match phases, and player performance.

⸻

Questions Answered

1. Does winning the toss help teams win more matches?
2. Which phase of the game is most important for winning?
3. Who are the top batters and bowlers across five seasons?

⸻

Dataset

Source:  Cricsheet IPL Data￼

The dataset contains:

* Batter and bowler details
* Runs and wickets
* Toss results
* Match winners
* Overs and innings

Each row represents one ball bowled.

⸻

Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

⸻

Analysis

Toss Impact

Compared toss winners and toss losers with actual match results.

Insight

Winning the toss gives only a small advantage.

⸻

Match Phase Analysis

Overs were divided into:

* Powerplay (0–5)
* Middle Overs (6–14)
* Death Overs (15–19)

Average runs scored by winning and losing teams were compared.

Insight

Powerplay scoring had the biggest impact on winning matches.

⸻

Player Performance

Batters

Ranked by total runs scored.

Bowlers

Ranked by total wickets taken.

Insight

A few players consistently performed well across multiple seasons.

⸻

Visualizations

Key Findings

* Toss impact is smaller than expected
* Powerplay performance strongly affects match results
* Consistent players dominate IPL statistics
